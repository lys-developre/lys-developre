## 🚀 Sobre mí

Soy un **Desarrollador Web Full Stack** apasionado por la creación de soluciones eficientes y escalables. Mi enfoque principal es el **Backend**, donde me especializo en construir infraestructuras sólidas y APIs seguras.

- 🛠️ Actualmente trabajando con **Laravel, Symfony y Next.js**.
- 🐳 Experto en contenedores con **Docker** y despliegue continuo (**CI/CD**).
- 🧪 Fiel creyente del **Testing automatizado** (PHPUnit) para asegurar la calidad.
- 👨‍💻 Metodologías ágiles (**Scrum**) para una entrega de valor constante.
